# installs

- Please place the GEMINI.md file as the goolden rule in ~/.gemini/GEMINI.md.

- And you can domain rule by workspace in <workspace>/.agent/rules/<RULE>.md

# Caution

- Regarding the "RULE" files, please ensure that the character count for each does not exceed 12,000.
